# ­ЪЈЏ№ИЈ RS InfoHub AI Agent (RAG)

**RS InfoHub AI** рЃљрЃарЃўрЃА рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃФрЃўрЃћрЃЉрЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (RAG - Retrieval-Augmented Generation), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃцрЃўрЃюрЃљрЃюрЃАрЃЌрЃљ рЃАрЃљрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃЮрЃА рЃерЃћрЃЏрЃЮрЃАрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃАрЃљрЃ«рЃБрЃарЃўрЃА (RS) рЃАрЃљрЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃЮ рЃ░рЃљрЃЉрЃўрЃА (`infohub.rs.ge`) рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ.

рЃљрЃњрЃћрЃюрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **Hybrid Search** рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљрЃА (BM25 + Semantic Vector Search), рЃарЃљрЃЌрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃА рЃњрЃљрЃАрЃфрЃћрЃА рЃќрЃБрЃАрЃбрЃў рЃърЃљрЃАрЃБрЃ«рЃў рЃЊрЃљ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃў.

## ­Ъџђ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

*   **Hybrid Retrieval:** рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃАрЃўрЃбрЃДрЃЋрЃљрЃАрЃўрЃбрЃДрЃЋрЃўрЃЌ рЃФрЃўрЃћрЃЉрЃљрЃА (BM25) рЃЊрЃљ рЃерЃўрЃюрЃљрЃљрЃарЃАрЃЮрЃЉрЃарЃўрЃЋ рЃФрЃўрЃћрЃЉрЃљрЃА (Vector Search / Embeddings) рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА.
*   **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃћрЃюрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ:** рЃАрЃарЃБрЃџрЃљрЃЊ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃЊрЃљ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ-рЃърЃљрЃАрЃБрЃ«рЃќрЃћ.
*   **рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ:** рЃърЃљрЃАрЃБрЃ«рЃА рЃЌрЃљрЃю рЃћрЃарЃЌрЃЋрЃўрЃА рЃгрЃДрЃљрЃарЃЮрЃА рЃџрЃўрЃюрЃЎрЃћрЃЉрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃерЃўрЃю, рЃЌрЃБ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃЌ рЃарЃћрЃџрЃћрЃЋрЃљрЃюрЃбрЃБрЃарЃЮрЃЉрЃљрЃерЃў (Threshold-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ).
*   **Smart Filtering:** рЃцрЃўрЃџрЃбрЃарЃљрЃЋрЃА "Smalltalk"-рЃА (рЃЏрЃљрЃњ: "рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ", "рЃарЃЮрЃњрЃЮрЃа рЃ«рЃљрЃа"), рЃарЃљрЃЌрЃљ рЃљрЃа рЃЊрЃљрЃ«рЃљрЃарЃ»рЃЮрЃА рЃарЃћрЃАрЃБрЃарЃАрЃў рЃЉрЃљрЃќрЃўрЃА рЃФрЃўрЃћрЃЉрЃљрЃќрЃћ.
*   **Debug Mode:** рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃарЃћрЃЪрЃўрЃЏрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃФрЃўрЃћрЃЉрЃўрЃА рЃАрЃЦрЃЮрЃарЃћрЃЉрЃА (Scores), рЃЏрЃћрЃбрЃљрЃЊрЃљрЃбрЃљрЃА рЃЊрЃљ рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃА.

## РџЎ№ИЈ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ:

# рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

# рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃљ (Windows)

# рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃљ (macOS/Linux)

## ­Ъћё рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (Data Pipeline)

рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃЏрЃЊрЃћ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃљрЃфрЃўрЃљ.

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ
рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ RS InfoHub API-рЃА рЃЊрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА `data/raw_docs.jsonl` рЃцрЃљрЃўрЃџрЃерЃў.

```bash
python src/download_data.py
```
*рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃърЃарЃЮрЃњрЃарЃћрЃАрЃА `state.json`-рЃерЃў. рЃерЃћрЃгрЃДрЃЋрЃћрЃбрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃњрЃљрЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ рЃўрЃЦрЃўрЃЊрЃљрЃю, рЃАрЃљрЃЊрЃљрЃф рЃњрЃљрЃЕрЃћрЃарЃЊрЃљ.*

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2: рЃўрЃюрЃЊрЃћрЃЦрЃАрЃљрЃфрЃўрЃљ (Ingestion)
рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА HTML рЃбрЃћрЃЦрЃАрЃбрЃА, рЃГрЃарЃўрЃА рЃЏрЃљрЃА рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљрЃЊ (Chunks), рЃЦрЃЏрЃюрЃўрЃА рЃЋрЃћрЃЦрЃбрЃЮрЃарЃћрЃЉрЃА (ChromaDB) рЃЊрЃљ BM25 рЃўрЃюрЃЊрЃћрЃЦрЃАрЃА (`chunks.jsonl`).

```bash
python src/ingest.py
```

## ­ЪДа рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Hybrid Search?

рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЮрЃарЃў рЃбрЃўрЃърЃўрЃА рЃФрЃўрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃџрЃљрЃЊ:

1.  **BM25 (Keyword Search):** рЃћрЃФрЃћрЃЉрЃА рЃќрЃБрЃАрЃб рЃЌрЃљрЃюрЃ«рЃЋрЃћрЃЊрЃарЃћрЃЉрЃА (рЃЏрЃљрЃњ: рЃЎрЃљрЃюрЃЮрЃюрЃўрЃА рЃюрЃЮрЃЏрЃарЃћрЃЉрЃў, рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў - "рЃЏрЃБрЃ«рЃџрЃў 153").
2.  **Vector Search (Semantic):** рЃћрЃФрЃћрЃЉрЃА рЃљрЃќрЃарЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃЏрЃАрЃњрЃљрЃЋрЃА рЃбрЃћрЃЦрЃАрЃбрЃћрЃЉрЃА (рЃЏрЃљрЃњ: "рЃарЃЮрЃЊрЃўрЃА рЃЋрЃўрЃ«рЃЊрЃў рЃЊрЃдрЃњ-рЃА?").

**рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў:**
1.  рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃўрЃдрЃћрЃЉрЃА `Top-K` рЃерЃћрЃЊрЃћрЃњрЃА BM25-рЃўрЃАрЃњрЃљрЃю.
2.  рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃўрЃдрЃћрЃЉрЃА `Top-K` рЃерЃћрЃЊрЃћрЃњрЃА ChromaDB-рЃАрЃњрЃљрЃю.
3.  рЃ«рЃЊрЃћрЃЉрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃћрЃЉрЃўрЃА рЃгрЃљрЃерЃџрЃљ.
4.  LLM (Gemini) рЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃърЃљрЃАрЃБрЃ«рЃА рЃюрЃљрЃърЃЮрЃЋрЃюрЃў рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ.

## ­ЪјЏ№ИЈ рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃърЃљрЃюрЃћрЃџрЃў (Sidebar)

рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃарЃфрЃ«рЃћрЃюрЃљ рЃърЃљрЃюрЃћрЃџрЃўрЃЊрЃљрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃЮрЃЌ:

*   **Debug рЃарЃћрЃЪрЃўрЃЏрЃў:** рЃЕрЃљрЃарЃЌрЃЋрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃърЃљрЃАрЃБрЃ«рЃўрЃА рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў (Retrieval Scores, рЃюрЃљрЃърЃЮрЃЋрЃюрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА ID-рЃћрЃЉрЃў рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃюрЃљрЃгрЃДрЃЋрЃћрЃбрЃћрЃЉрЃў).
*   **Top-K рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў:** рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃгрЃљрЃЏрЃЮрЃўрЃдрЃЮрЃА рЃЉрЃљрЃќрЃўрЃЊрЃљрЃю (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ 4-6).
*   **Threshold (рЃќрЃдрЃЋрЃљрЃарЃў):** рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЏрЃАрЃњрЃљрЃЋрЃАрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃћрЃцрЃўрЃфрЃўрЃћрЃюрЃбрЃў (0.0 - 1.0). рЃЌрЃБ рЃюрЃљрЃърЃЮрЃЋрЃюрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃЦрЃЮрЃарЃў рЃљрЃЏ рЃќрЃдрЃЋрЃљрЃарЃќрЃћ рЃЊрЃљрЃЉрЃљрЃџрЃўрЃљ, рЃгрЃДрЃљрЃарЃЮрЃА рЃџрЃўрЃюрЃЎрЃў рЃљрЃа рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ, рЃарЃљрЃЌрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃа рЃерЃћрЃЋрЃўрЃДрЃЋрЃљрЃюрЃЮрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃерЃў.

## ­ЪЉеРђЇ­Ъњ╗ рЃљрЃЋрЃбрЃЮрЃарЃў
рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ InfoHub-рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃЊрЃћрЃА рЃАрЃљрЃњрЃљрЃЊрЃљрЃАрЃљрЃ«рЃљрЃЊрЃЮ рЃЊрЃљ рЃАрЃљрЃЉрЃљрЃЪрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃљ.
